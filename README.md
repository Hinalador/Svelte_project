# Habit Tracker (Svelte)

Una aplicación sencilla para el seguimiento de hábitos diarios construida con [Svelte](https://svelte.dev/) y [Vite](https://vitejs.dev/).

## Características

- Lista de hábitos diarios (e.g. Correr 5k, Leer, Meditar).
- Indicadores visuales y alertas dinámicas según el progreso (inicio, en progreso, éxito).
- Barra de porcentaje y conteo de hábitos completados.
- Botón para reiniciar el progreso para el comienzo de un nuevo día.

## Tecnologías Utilizadas

- **Svelte** - Reactividad moderna y sencilla para interfaces de usuario.
- **Vite** - Servidor de desarrollo extremadamente rápido.
- **HTML, CSS (Vanilla), JavaScript**

## Instalación y Ejecución Local

1. Clona este repositorio en tu computadora:
   ```bash
   git clone https://github.com/Hinalador/Svelte_project.git
   ```

2. Entra al directorio del proyecto:
   ```bash
   cd Svelte_project/habit-tracker
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

4. Ejecuta el servidor de desarrollo local:
   ```bash
   npm run dev
   ```

5. Abre [http://localhost:5173/](http://localhost:5173/) en tu navegador para ver y utilizar la aplicación.
